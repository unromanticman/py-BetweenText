# py-BetweenText
Python get text between two word.  
If you want to get text between two word,you will use it.

## Spport
Python 2.7

## Usage
<pre>
from BetweenText import BetweenText

text =  BetweenText.get('asd3get this text1231','asd3','1231')
print text
</pre>
